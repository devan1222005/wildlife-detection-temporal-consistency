# Livestock Farm Protection from Harmful Wild Animals Using YOLOv11 with Temporal Consistency

Real-time wildlife intrusion detection and livestock farm protection system using YOLOv11, temporal consistency, OpenCV, Flask, and deep learning techniques.

## Features
- Real-time wild animal detection
- YOLOv11-based object detection
- Temporal consistency filtering
- Reduced false alarms
- Email alert generation
- Video-based monitoring system
- Flask web application interface

## Technologies Used
- Python
- YOLOv11
- OpenCV
- Flask
- NumPy
- Deep Learning
- Computer Vision

## Project Workflow
1. Upload surveillance video
2. Detect harmful wild animals
3. Apply temporal consistency filtering
4. Generate intrusion alerts
5. Display detection results

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app.py
```

## Results
- Real-time wildlife detection
- Reduced false positives using temporal consistency
